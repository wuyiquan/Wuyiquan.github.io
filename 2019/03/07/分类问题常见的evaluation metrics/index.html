分类问题常见的evaluation metrics

把结果分为四种

  1. 本类样本被分到了本类中 TP
  2. 本类样本被分到了错误类 FP
  3. 其他类样本被分到了本类 FN
  4. 其他类样本被分到了其他类 TN

  • 准确率（Accuracy）

  • 精确率（Precision)

  • 召回率 （Recall）

  • F-measure

通常Beta取1，即$$F_{1}$$。

##多类的情况

multilabel 或者 multiclass 时，会用到宏平均和微平均。
